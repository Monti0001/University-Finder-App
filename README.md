# 🌍 University Finder App

A simple web app that allows users to search for universities by country using a public API.

## 🚀 Features
- 🔍 Search universities by country name
- 🌐 Uses real-time API data
- ⚡ Fast and responsive UI
- 📃 Displays list of universities dynamically

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- Axios (for API requests)

## 📡 API Used
https://universities.hipolabs.com/

## 💻 How It Works
1. User enters a country name
2. Clicks on the search button
3. App fetches data from API
4. Displays universities in a list format

## ▶️ Run Locally
1. Clone the repo
2. Open `index.html` in browser
